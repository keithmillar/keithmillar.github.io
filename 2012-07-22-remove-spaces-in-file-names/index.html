Remove Spaces in File Names

Jul 22, 2012
computing linux

Removes spaces in file names and replaces them with

{% highlight bash html %} for file in ; do mv “$file” echo $file | sed -e 's/ */-/g' -e 's/_-_/-/g'; done {% endhighlight %} or {% highlight bash html %} find . -depth -name ‘ *’ | while IFS= read -r f ; do mv -i “$f” “$(dirname “$f”)/$(basename “$f”|tr ‘ ’ _)” ; done {% endhighlight %}
